# 🗺️ LaporanApp

Aplikasi mobile pelaporan masalah lingkungan berbasis sosial media, dibangun dengan React Native & Expo.

## 📱 Fitur Utama

- **Home** — Peta interaktif lokasi pengguna + tombol buat laporan
- **Leaderboard** — Ranking kontributor terbanyak
- **Forum** — Feed laporan masyarakat dengan fitur sort & upvote
- **Detail Laporan** — Foto, deskripsi, alamat, dan komentar
- **Task & Kupon** — Tukar upvote menjadi voucher reward

## 🛠️ Teknologi

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Expo Router](https://expo.github.io/router/)
- [React Native Maps](https://github.com/react-native-maps/react-native-maps)
- [Expo Image Picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/)

## 🚀 Cara Menjalankan

1. Clone repository ini
   
   git clone https://github.com/USERNAME/LaporanApp.git

2. Masuk ke folder project
   
   cd LaporanApp

3. Install dependencies
   
   npm install

4. Jalankan app
   
   npx expo start

5. Scan QR code menggunakan **Expo Go** di HP

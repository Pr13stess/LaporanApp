# MariLapor! 📣

> **Infrastruktur Kita, Tanggung Jawab Kita**

Aplikasi mobile pelaporan masalah lingkungan & infrastruktur berbasis sosial media, dibangun dengan React Native & Expo.

---

## 📱 Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| 🗺️ **Home** | Peta interaktif lokasi pengguna & tombol buat laporan |
| 🏆 **Leaderboard** | Ranking kontributor terbanyak |
| 💬 **Forum** | Feed laporan masyarakat dengan fitur sort & upvote |
| 📄 **Detail Laporan** | Foto, deskripsi, alamat, dan komentar |
| 🎁 **Task & Kupon** | Tukar upvote menjadi voucher reward |

---

## 🛠️ Teknologi

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Expo Router](https://expo.github.io/router/)
- [Supabase](https://supabase.com/) — Auth & Database
- [React Native Maps](https://github.com/react-native-maps/react-native-maps)
- [Expo Image Picker](https://docs.expo.dev/versions/latest/sdk/imagepicker/)

---

## 🚀 Cara Menjalankan

```bash
# 1. Clone repository
git clone https://github.com/USERNAME/MariLapor.git

# 2. Masuk ke folder project
cd MariLapor

# 3. Install dependencies
npm install

# 4. Jalankan app
npx expo start
```

Scan QR code menggunakan **Expo Go** di HP kamu.
